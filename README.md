# ClashX
ClashX，ClashX教程，ClashX配置教程，ClashX for mac  



简介
----

Clash 是一个使用 Go 语言编写，基于规则的跨平台代理软件核心程序。  
ClashX 是运行在 macOS 上的一款科学上网工具。  

**支持的协议:**  
 Vmess  
 Shadowsocks  
 Snell  
 SOCKS5  
 
 
 **特色**  
- 可随时切换代理模式及节点  
- 支持节点批量延迟测试  
- 通过托管链接一键配置  
- 规则命中分析  
- 日志输出  


**设备要求**  

系统要求：macOS 10.5 及以上   

设备要求：MacBook / iMac / Mac Pro
